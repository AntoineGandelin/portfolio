Application - CDI

Application web pour un collège


Contexte : Stage de première année


Date de début : 31 mai 2021

Date de début : 02 juilet 2021

Durée de la réalisation professionnelle : 5 semaines


Description

J'ai réalisé mon stage de première année de BTS SIO au collège Marie-Hélène Cardot situé à Douzy dans les Ardennes.
Ma mission consistait à réaliser une application web permettant de faire l'appel des élèves au CDI du collège.


Informations

OS utilisé(s) Outil(s) utilisé(s) Langage(s) utilisé(s)
Windows 10 Visual Studio Code, Wamp Server, PhpPgAdmin PHP, PostgreSQL, HTML, CSS
